# Voice Agent Web — Backend

FastAPI backend for the LiveKit voice agent web app. Handles signed JWT verification, LiveKit token generation, and agent dispatch.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/agents` | Returns the list of available agents from `LIVEKIT_AGENTS` env var |
| `POST` | `/connect` | Verifies a signed config token, issues a LiveKit room token, and dispatches the agent |

### `GET /agents`

Returns the statically configured agent list:
```json
[{"id": "voice-agent", "name": "Alex"}, ...]
```

### `POST /connect`

Takes a signed JWT (generated by `generate_call_url.py` on the agent host) and does the full room setup in one call.

Request:
```json
{"config_token": "<signed-jwt>"}
```

Response:
```json
{
  "agent": {"id": "voice-agent-abc123", "name": "Alex"},
  "token": "<livekit-room-token>",
  "url": "wss://your-project.livekit.cloud",
  "room_name": "room-abc123def456",
  "dispatch_id": "..."
}
```

Error responses:
- `401` — expired or invalid config token
- `400` — missing `agent_name` in token payload
- `503` — `CONFIG_SECRET` not configured on the server

## Environment variables

| Variable | Required | Description |
|----------|----------|-------------|
| `LIVEKIT_URL` | ✅ | LiveKit server WebSocket URL (`wss://...`) |
| `LIVEKIT_API_KEY` | ✅ | LiveKit API key |
| `LIVEKIT_API_SECRET` | ✅ | LiveKit API secret |
| `CONFIG_SECRET` | ✅ | Shared signing secret for JWT URL verification (see below) |
| `LIVEKIT_AGENTS` | — | JSON array of available agents for the home screen dropdown |
| `CORS_ORIGINS` | — | Comma-separated allowed origins (default: `*`) |

## CONFIG_SECRET — the shared signing secret

`CONFIG_SECRET` is used to sign and verify JWT call URLs. It must be the same value in:
1. The agent host `.env` (used by `generate_call_url.py` to sign URLs)
2. The Fly.io backend secret (used by `/connect` to verify URLs)

Generate a new secret:
```bash
python3 -c "import secrets; print(secrets.token_hex(32))"
```

Set it on Fly:
```bash
fly secrets set CONFIG_SECRET=<your-secret>
```

## Development

```bash
cp .env.example .env
# fill in your values, including CONFIG_SECRET

pip install -e .
uvicorn main:app --reload --port 8000
```

## Deployment (Fly.io)

```bash
# First time
fly launch --no-deploy

# Set secrets (CONFIG_SECRET must match the agent host .env)
fly secrets set \
  LIVEKIT_URL=wss://... \
  LIVEKIT_API_KEY=... \
  LIVEKIT_API_SECRET=... \
  CONFIG_SECRET=... \
  LIVEKIT_AGENTS='[{"id":"voice-agent","name":"Alex"}]'

fly deploy
```
